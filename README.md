# 🛒 Mini E-commerce React

Un projet de boutique en ligne moderne développé avec **React.js** et **Tailwind CSS**, permettant d’afficher des produits, filtrer par catégorie et gérer un panier d’achat.

---

## 🚀 Fonctionnalités

- 🏠 Affichage des produits
- 🔎 Filtrage par catégorie (Phone, Laptop, Audio)
- 📄 Page détail produit
- 🛒 Système de panier (add / remove)
- 💰 Calcul du total automatique
- ❌ Page panier vide
- 🎨 Interface moderne et responsive
- 💾 Sauvegarde du panier (localStorage)

---

## 🛠️ Technologies utilisées

- React.js
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)
